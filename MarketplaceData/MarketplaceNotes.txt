= Name (en) =
WinMilk Task Manager

= Detailed Description (en) =

WinMilk is a task manager and to-do list app that syncs with your Remember the Milk account. WinMilk lets you view your RTM tasks by date, list, smart list and tag, as well as search. WinMilk also lets you add and edit tasks and notes. The app caches data locally so you can see your tasks even if you do not currently have a network connection.

New in 1.3: Note adding and editing (tap and hold a note to edit), landscape mode, sharing by SMS/email, autocorrect, creating recurring tasks, and LOADS of bug fixes.

For more information on WinMilk, visit http://julianapena.com/projects/winmilk/

WinMilk is an open source project at http://winmilk.codeplex.com/

= Name (es) =
WinMilk Tareas

= Detailed Description (es) =

WinMilk es una aplicación para administrar tareas que se sincroniza con Remember the Milk. WinMilk te deja ver tus tareas for fecha, lista, etiqueta y búsqueda. Además, puedes agregar y editar tareas y notas. La aplicación usa descarga tus tareas para que las puedas ver incluso si no tienes conexión a red.

Nuevo en 1.3: Adición y edición de notas (mantén pulsada una nota para editarla), modo horizontal, compartir tareas por SMS/email, crear tareas recurrentes, y MUCHOS errores corregidos.

Para más información sobre WinMilk, visita http://julianapena.com/projects/winmilk/

WinMilk es un proyecto código abierto en http://winmilk.codeplex.com/
